# Contributing

Thanks for helping improve Short Video AI Studio.

## Local Setup

```bash
make install
make dev
```

Open `http://127.0.0.1:8000`.

## Checks

Run this before opening a pull request:

```bash
make check
```

## Notes

- Do not commit `.env`, user uploads, generated output, or local databases.
- Keep prompts and UI copy focused on short-video creators.
- Prefer small, reviewable changes.
